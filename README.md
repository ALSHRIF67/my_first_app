## 🛠️ Technologies Used

* Flutter
* Dart

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/ALSHRIF67/my_first_app.git
```

Go to the project folder:

```bash
cd my_first_app
```

Install dependencies:

```bash
flutter pub get
```

Run the app:

```bash
flutter run
```
